# GCS&I COE 2026 First Half Recap

## Publish publicly with GitHub Pages

1. Create a new public GitHub repository.
2. Upload `index.html`.
3. Open **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root`, then save.
6. GitHub will provide the public website URL.

## Optional background music

Add a royalty-free MP3 named `music.mp3` into the same folder as `index.html`.
The music button will then work automatically.
